# DDOpenKit
an ios tool lib

## 通过cocoaPods安装

```shell
pod 'DDOpenKit', '~> 1.1.3'
```

在需要的地方引入头文件

```objective-c
#import <DDOpenKit/DDOpenKit.h>
```



 收集了很多实用的适合单人开发 mvc的工具集库
